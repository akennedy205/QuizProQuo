## Installation and Usage
### Installation
 * Clone this repo
 * `cd` into server file and in terminal, `npm install`
 * `cd` into frontend file and in terminal, `npm install`
### Usage
 * For client, `cd frontend` and `npm run start`
 * For server, `cd server` and `npm run dev`
